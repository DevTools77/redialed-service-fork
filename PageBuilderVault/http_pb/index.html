WebTV community home pages
Here you'll find pages created by members of the WebTV community using WebTV's Page Builder.
If you know the name of a WebTV user who has a home page, you can find that user's home page at an address of this form:

http://community.webtv.zone/username
For example, if the username is ltolstoy, then the address is:

http://community.webtv.zone/ltolstoy
